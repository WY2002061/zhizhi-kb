# 智知 — AI 个人知识库

一个基于 AI 的个人知识管理 Web 应用，帮助用户聚合、整理、回顾个人知识。

## 功能特性

- **智能收藏** — 支持网页链接、文本、笔记等多种来源，AI 自动生成摘要和标签
- **知识卡片** — 每条知识以卡片形式呈现，支持收藏、批注、隐私分级
- **知识空间** — 自定义知识集分类，按主题管理知识
- **智能搜索** — 自然语言语义搜索，支持标签和知识集筛选
- **回顾中心** — 每日简报、间隔复习（基于遗忘曲线）、随机漫游三种复习模式
- **知识图谱** — 力导向图可视化知识之间的关联网络
- **隐私管理** — 四层隐私架构（公开 / 私密 / 机密），敏感信息自动检测与弹窗确认，完整审计日志
- **数据导入/导出** — 支持 JSON 格式的数据导出与导入，增量合并不覆盖已有数据
- **批量操作** — 多选知识卡片后可批量删除、移动知识集、变更隐私级别
- **新手引导** — 首次使用时自动弹出 6 步引导教程，快速了解核心功能
- **双主题** — V1 米黄色暖色调 / V2 莫兰迪极简风，一键切换

## 技术栈

| 层面 | 技术 |
|------|------|
| 框架 | React 19 + TypeScript |
| 构建 | Vite 8 |
| 样式 | Tailwind CSS 4 |
| 状态管理 | Zustand |
| 路由 | React Router 7 |
| 图标 | Lucide React |
| 数据持久化 | localStorage（本地存储，无需后端） |

## 环境要求

- [Node.js](https://nodejs.org/) >= 18（推荐 20+）
- npm >= 9（随 Node.js 一起安装）

## 快速开始

```bash
# 1. 克隆仓库
git clone https://github.com/WY2002061/zhizhi-kb.git
cd zhizhi-kb

# 2. 安装依赖
npm install

# 3. 启动开发服务器
npm run dev
# 浏览器打开终端提示的地址（默认 http://localhost:5173）

# 4. 构建生产版本
npm run build

# 5. 本地预览构建结果
npm run preview
```

## 隐私设计

所有数据存储在用户本地浏览器中，不会上传至任何服务器。AI 分析功能以模拟方式运行，不会将用户内容发送到外部 API。敏感信息（密码、身份证号、手机号等）会被自动检测，用户可选择加密保存、脱敏处理或跳过。

## 数据管理

- **导出**：在「隐私管理」页面点击「导出数据」，生成 JSON 文件。机密内容自动脱敏。
- **导入**：点击「导入数据」选择之前导出的 JSON 文件，系统会增量合并新数据，不覆盖已有条目。
